添加mysql yum源

  • yum localinstall http://dev.mysql.com/get/mysql57-community-release-el7-7.noarch.rpm

安装mysql5.7

  • yum install mysql-community-server -y

查看版本信息

  • mysqld --version

常用命令:

开机自起: systemctl enable mysqld.service

启动:    systemctl start mysqld.service

停止:    systemctl stop mysqld.service

停止:    systemctl stop mysqld.service

修改初始密码

获取初始密码：grep 'temporary password' /var/log/mysqld.log

进入命令行： mysql -uroot -p

修改初始密码：ALTER USER 'root'@'localhost' IDENTIFIED BY 'newpwd';

添加远程连接帐号

  • GRANT ALL PRIVILEGES ON *.* TO 'newname'@'%' IDENTIFIED BY 'newpwd' WITH GRANT OPTION;

  • 在阿里云安全组给3306端口配置准入规则，配置授权对象为：0.0.0.0/0

  • 在可视化工具中host为服务器公网IP， 端口3306

修改编码方式

  • vim /etc/my.cnf

  • 新增并保存：

    	character-set-server=utf8
	init_connect='SET NAMES utf8

  • 重启mysql服务
