ssh免密登陆

1.清除ssh缓存

  • 远程连接服务器: ssh -p port name@ip

  • 清除ssh缓存 ssh-keygen -R ip

2.设置远程免密登录:

  • 用scp将本地公钥传到服务器: scp .ssh/id_rsa.pub root@ip:/root
  • 将公钥放到.ssh/authorized_keys文件夹下: cat id_rsa.pub >> .ssh/authorized_keys
  • 更改权限: chmod 600 .ssh/authorized_keys
  • 修改sshd配置: vim /etc/ssh/sshd_config （PubkeyAuthentication no -> yes）
  • 登陆服务器: ssh root@ip

3.关于Linux权限(数字权表示法):

  1. 数字4、2、1分别代表读、写、执行, 即: r=4 w=2 x=1。
  2. 按照用户身份可分成: 拥有则、群组、其他组。即: u、g、o
  3. 对应用户为需要什么权限相加即可: rw=6、rwx=7
  4. 使用命令行给某个文件夹/文件赋予三种身份对应的权限: chomd xxx fileName
