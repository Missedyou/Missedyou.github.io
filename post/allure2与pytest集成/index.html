allure2 与 pytest集成

Nov 16, 2018 14:20 · 29 words · 1 minute read python-base

    • 环境准备
    • 测试过程
    • 需要改进

环境准备

  • pip install allure-pytest
  • pip install pytest-allure-adaptor

如果安装了旧版本则需要先卸载后重新安装。

测试过程

执行脚本: pytest [your file or directory] –alluredir=[your result directory]

生成测试结果H5页面: allure serve ./[your result directory]

每次生成报告页面都需要敲两次命令, 建议写一份sh或者python脚本进行执行。

需要改进

每次测试都会保留上一次的结果, 打算用shell或者python脚本实现删除上一次记录、执行测试命令、生成测试报告。或者再看看官网有没有提供特殊的API
